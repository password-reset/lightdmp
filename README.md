# lightdmp
Dumps the currently logged on users' password from lightdm --session-child process memory.

```
root@pwnd# ./lightdmp.sh 
 _ _       _     _   ____  __  __ ____  
| (_) __ _| |__ | |_|  _ \|  \/  |  _ \ 
| | |/ _` | '_ \| __| | | | |\/| | |_) |
| | | (_| | | | | |_| |_| | |  | |  __/ 
|_|_|\__, |_| |_|\__|____/|_|  |_|_|    
     |___/                           
USERNAME: root 
PASSWORD: r00tp@ass89!
```
